Requirements:
	Windows 7 or 10
	Java 7 64-bit
	MySQL Server 5.7

Instructions:
1. Clone the project (MiningResourceProfilesPlugin)
2. Create system variable R_HOME with the value:
[Project path]\MiningResourceProfilesPlugin\JRI_DIR\R-3.0.0
3. Add to system Path variable the value: 
[Project path]\MiningResourceProfilesPlugin\JRI_DIR\jrilib
4. Open the project in your preferred JDE (e.g. Eclipse) and run Java Application ‘ProM with UITopia (MiningResourceProfilesPlugin)’
5. After ProM is launched, upload XES event log and start plugin ‘Mining Resource Profiles’
6. After the plugin is started, in the first configuration screen specify host name, user name and password for your MySQL Server and specify database name for the project (a new MySQL database with the specified name will be created)
